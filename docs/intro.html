Excel BI workbooks are not just code.

A real workbook can contain formulas, VBA, Power Query M, Power Pivot measures, CUBE references, named ranges, external links, buttons, hidden process sheets, and delivery-only copies. This project turns those risks into reusable agent workflows.

Safer workbook edits

Explicit workflows for formulas, VBA, buttons, hidden sheets, workbook links, and workbook structure.

BI-layer awareness

Specialist skills for Power Query, Power Pivot DAX, MDX/CUBE, ADO/SQL, and semantic model boundaries.

Delivery-ready output

Freeze formulas, remove dependencies, clean process sheets, and verify client-ready Excel files.

12 production skills for Excel BI agents.

Use the router for broad tasks, or invoke a specialist skill when the workbook layer is clear.

VBA and workbook automation
excel-vba-workbook-engineering exports/imports modules, checks buttons, preserves formulas, and validates Excel behavior where possible.
Power Query M
power-query-m-engineering handles query edits, refresh completion, dependency lineage, error localization, and performance timing.
Power Pivot / DAX / MDX
power-pivot-dax-modeling and mdx-cubevalue-extraction cover measures, contexts, members, CUBE formulas, and Data Model boundaries.
ADO / SQL access
excel-ado-sql-data-access covers workbook SQL, ADODB, ADOMD, provider strings, parameters, and recordset handling.
QA and delivery
excel-workbook-qa-auditor and excel-deliverable-publisher make risk review and clean `.xlsx` publishing repeatable.
Cross-agent distribution
Generated mirrors support Codex, Claude, and OpenCode from one canonical .agents/skills/ source.

Install once, reuse across agents.

Choose marketplace install for Codex users, local install for teams that want to inspect or customize, and manual scripts for controlled enterprise distribution.

Codex marketplace
codex plugin marketplace add 90le/microsoft-excel-bi-agent
codex plugin add microsoft-excel-bi-agent-pack@microsoft-excel-bi-agent
Local one-command install
git clone https://github.com/90le/microsoft-excel-bi-agent.git
cd microsoft-excel-bi-agent
node tools/install.mjs
The project does not advertise `npx` yet because no npm package has been published. Current stable one-command install is node tools/install.mjs.
