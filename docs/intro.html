Excel BI workbooks are not just code.

A real workbook can contain formulas, macros, query dependencies, Data Model measures, CUBE references, named ranges, external links, buttons, hidden process sheets, and delivery-only copies. This project gives agents a disciplined operating model for that world.

Safer workbook edits

Agents get explicit workflows for VBA, formulas, buttons, hidden sheets, links, and workbook structure.

BI-layer awareness

Power Query M, Power Pivot DAX, MDX/CUBE, ADO/SQL, and semantic model boundaries are routed to specialist skills.

Delivery-ready outputs

Publishing workflows freeze formulas, remove dependencies, clean process sheets, and verify client-ready Excel files.

12 production skills for Excel BI agents.

Use the router for broad tasks or invoke the specialist skill directly when the workbook layer is clear.

VBA and workbook automation
excel-vba-workbook-engineering exports/imports modules, checks buttons, preserves formulas, and validates Excel behavior where possible.
Power Query M
power-query-m-engineering handles query edits, refresh completion, dependency lineage, error localization, and performance timing.
Power Pivot / DAX / MDX
power-pivot-dax-modeling and mdx-cubevalue-extraction help agents reason about measures, contexts, members, CUBE formulas, and Data Model boundaries.
ADO / SQL access
excel-ado-sql-data-access covers workbook SQL, ADODB, ADOMD, provider strings, parameters, and recordset handling.
QA and delivery
excel-workbook-qa-auditor and excel-deliverable-publisher turn workbook risk review and clean `.xlsx` publishing into repeatable agent workflows.
Cross-agent distribution
Generated mirrors support Codex, Claude, and OpenCode from one canonical .agents/skills/ source.

Install once, reuse across agents.

Clone the repository, install as a local Codex plugin, then sync the same skill source to project mirrors and user-level skills.

git clone https://github.com/90le/microsoft-excel-bi-agent.git cd microsoft-excel-bi-agent
python tools\deploy-local-plugin.py --project-root . --replace --install python tools\sync-skills.py --project-root . --all-project-mirrors --codex-user --replace
MIT licensed. Windows desktop Excel is required for Excel COM, VBA execution, Power Query refresh, and Power Pivot runtime proof. macOS and Linux support structural checks and OpenXML inspection.
