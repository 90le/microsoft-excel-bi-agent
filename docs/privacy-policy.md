# Privacy Policy

Last updated: July 14, 2026

Microsoft Excel BI Agent is an open-source plugin that runs in the user's selected agent and local environment. The project maintainer does not operate a hosted service for this plugin and does not collect plugin telemetry, workbook contents, prompts, credentials, or generated outputs.

## Data Handling

- Files and prompts are processed by the agent, model provider, operating system, and tools chosen by the user. Their privacy policies and configuration govern that processing.
- The plugin is designed for local-file workflows. Users control which workbooks and outputs are made available to an agent.
- Customer workbooks, credentials, screenshots, generated reports, and private paths must not be committed to this public repository or attached to public issues.
- Sanitized fixtures should be used when reproducing a problem publicly.

## Third-Party Services

GitHub, Codex, Claude, OpenCode, Microsoft Excel, and other optional tools are third-party services or products. Use of them is subject to their own privacy terms.

## Contact

For a private security report, follow [SECURITY.md](../SECURITY.md). For general privacy questions, open a repository issue without including confidential data.

This policy may be updated when the plugin's data flows or distribution model change.
