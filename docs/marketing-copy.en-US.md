## Launch Post

Microsoft Excel BI Agent is an open-source Excel BI skill pack for AI agents that need to work on real Microsoft Excel workbooks.

It covers VBA, Power Query M, Power Pivot DAX, MDX/CUBE formulas, ADO/SQL, workbook QA, clean deliverables, Office diagnostics, report building, semantic model review, and sanitized testing fixtures.

Install with Codex Marketplace:

```bash
codex plugin marketplace add 90le/microsoft-excel-bi-agent
codex plugin add microsoft-excel-bi-agent-pack@microsoft-excel-bi-agent
```

Local install:

```bash
node tools/install.mjs
```

Windows desktop Excel is still required to prove Excel COM, VBA execution, Power Query refresh, and Power Pivot runtime behavior. macOS and Linux support structural checks and OpenXML inspection.

## Do Not Claim

- Do not claim a public npm or npx install path until it exists.
- Do not claim workbook-specific accuracy without reviewing that workbook.
- Do not claim Excel COM, VBA, Power Query refresh, or Power Pivot runtime proof from macOS/Linux structural checks.
- Do not use customer workbook screenshots or generated QA reports as proof.
